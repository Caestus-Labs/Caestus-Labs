Variant B (Balanced). Mark built on a 24-unit grid, 2u stroke.

## Type system

  Wordmark "CAESTUS" — Archivo, weight 800, tracking ~0.24em
  Sub-word "LABS"   — Inter,   weight 300, tracking 0.36em, 60% opacity

Both are free Google Fonts:
  https://fonts.google.com/specimen/Archivo
  https://fonts.google.com/specimen/Inter

## Usage rules

  Minimum size       16px (mark), 80px (horizontal lockup)
  Clear-space        1× stroke width on all sides of the mark
  Backgrounds        Always high-contrast. Use mono variants on photos.
  Don't              Recolor the mark. Stretch. Skew. Rotate. Add shadows.

## A note on the SVG text

The lockup SVGs reference Archivo and Inter via @import. Browsers render
these correctly. When importing into Figma/Illustrator/Inkscape, the app
will prompt to install the fonts (both free) — install them and the
wordmark renders identically. For absolute portability (no font needed
at all), open the lockup SVG in Illustrator and convert text to outlines.
